# Domain Map — Todo

**Version:** 1.0.0
**Last updated:** 2026-06-20

> Recommended, graph-assisted: these domains line up with the graph's communities after a sync.

---

## Tasks

**Responsibility:** Creating, completing, editing, and scheduling individual tasks.
**Owner:** Core team
**Status:** evolving
**Repos:** todo-api, todo-web

**Core features**
- Create a task (with optional due date)
- Mark a task complete / reopen it
- Edit and soft-delete a task

**Out of scope:** sharing tasks between users, recurring tasks.

**Key terms:** Task, Due date

---

## Lists

**Responsibility:** Grouping tasks into named lists and moving tasks between them.
**Owner:** Core team
**Status:** stable
**Repos:** todo-api, todo-web

**Core features**
- Create and rename a list
- Move a task into a list
- Reorder lists

**Out of scope:** nested lists, list templates.

**Key terms:** List
